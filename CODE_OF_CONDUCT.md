# Code of Conduct

This project, as part of the Ubuntu and Canonical community, follows the
[Ubuntu Code of Conduct](https://ubuntu.com/community/ethos/code-of-conduct).

We expect everyone participating in the project — through issues, pull requests,
discussions, or any other channel — to abide by it. It sets out the standards of
respectful, considerate, and collaborative behaviour we hold ourselves and each
other to.

If you experience or witness behaviour that violates the Code of Conduct, please
report it to the Canonical community team as described in the
[Ubuntu Code of Conduct](https://ubuntu.com/community/ethos/code-of-conduct).
